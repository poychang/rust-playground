## 更新 Rust

`rustup update`

## cargo 基本指令

詳請參考官方 [The Cargo Book](https://doc.rust-lang.org/cargo/index.html) 文件

- `cargo --version` 查看版本
- `cargo build` 能建置您的專案
- `cargo run` 能執行您的專案
- `cargo test` 能測試您的專案
- `cargo doc` 能為您的專案產生技術文件
- `cargo publish` 能將函式庫發佈到 crates.io
